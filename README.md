# adriennekruzer.github.io

Personal website published with [GitHub Pages](https://pages.github.com/).

The homepage hosts the **NPF Cultivate Matrix** — an interactive view of domains
by contribution level, with subdomain expansion, search, level/capability
filters, task detail modals, and a gaps tab.

Live at https://adriennekruzer.github.io

## Editing

The whole site is a single self-contained `index.html` (styles + markup + script
inline, no build step). Edit it, then:

```
git add -A && git commit -m "Update site" && git push
```

GitHub Pages redeploys automatically within a minute.
